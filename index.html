Using The App Offline

This app is a Progressive Web App (PWA), which means you can install it on your phone to use it offline, just like a regular app!

On Android (Chrome):

Look for an "Install" button in the address bar, or tap the three-dot menu and select "Install app".

On iPhone/iPad (Safari):

Tap the "Share" button, then scroll down and select "Add to Home Screen".
